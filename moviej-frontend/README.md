# myHealth-frontend

이 프로젝트는 Next.js 기반의 건강 관리 웹 프론트엔드입니다.

## 시작 방법

1. 개발 서버 실행:
   ```bash
   npm run dev
   ```
2. 브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

## 주요 기술

- Next.js 14
- React 18
- TypeScript
- Sass

## 개발/배포

- 코드 수정 시 자동 반영
- Vercel 등 다양한 플랫폼에 쉽게 배포 가능
